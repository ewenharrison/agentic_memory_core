# Autonomous Lane Policy

## Status
Enabled or disabled, with date if known.

## Permission
State whether the user has granted permission for Tier 2 to be used proactively for exploratory project maintenance and synthesis.

## What The Agent May Add Here
- candidate bibliographies
- rough literature maps
- speculative connections between sources
- draft taxonomies
- temporary synthesis notes
- search traces and working summaries

## What Tier 2 Must Not Assume
- Material in `auto/` is not canonical by default.
- Claims in `auto/` should not be treated as approved unless promoted to `approved/`.
- Tier 2 can inform `memory.md`, but durable claims should remain clearly marked as provisional until reviewed.

## Promotion Rule
Anything especially useful can later be promoted into `approved/` after review.
